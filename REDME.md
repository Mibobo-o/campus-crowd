# Campus Crowd
캠퍼스 혼잡도 예측 지도 프로젝트

## 색상 규칙 (Occupancy Level)
- <30% = Level 0 (초록)
- <60% = Level 1 (노랑)
- <80% = Level 2 (주황)
- ≥80% = Level 3 (빨강)
